Analysis:

The issue is the negative margin (-mx-4) on the category tabs container combined with the Fragment wrapper.

At 375px width:

  • Container width: 375px
  • Main padding (px-4): 16px on each side = 32px total
  • Content area: 375px - 32px = 343px
  • Negative margin (-mx-4): -16px on each side
  • Tabs container width: 343px + 32px = 375px (extends to viewport edges)

The negative margins are meant to make the scrollable tabs extend edge-to-edge on mobile, breaking out of the main content padding.
